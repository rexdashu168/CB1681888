## 📦 檔案清單

### 必要檔案（兩個）
1. **kb_index.html** - 知識庫系統主程式
2. **kb_data_complete.json** - 完整知識庫資料（19篇文章）

### 檔案說明
- **kb_index.html**：前端介面，包含所有功能
- **kb_data_complete.json**：整合所有批次的完整資料庫

---

## 🚀 快速開始

### 步驟1：下載檔案
```
從 outputs 資料夾下載：
✓ kb_index.html
✓ kb_data_complete.json
```

### 步驟2：放在同一資料夾
```
📁 任意資料夾名稱（例如：Rex知識庫）
  ├── kb_index.html
  └── kb_data_complete.json
```

### 步驟3：開啟使用
```
雙擊 kb_index.html
→ 瀏覽器會自動開啟
→ 開始使用！
```

**⚠️ 重要：兩個檔案必須在同一個資料夾！**

## ✨ 系統功能介紹

### 🔍 智能搜尋
- **全文搜尋**：搜尋標題、內容、關鍵字
- **即時過濾**：輸入關鍵字立即顯示結果
- **支援中文**：完美支援繁體中文搜尋

**使用範例：**
```
搜尋「競拍」→ 找到所有競拍相關文章
搜尋「融資」→ 找到所有融資風險文章
搜尋「陽明」→ 找到陽明五案例
```

### 📂 分類瀏覽
**8大分類：**
1. 基礎觀念（6篇）
2. 投資策略（2篇）
3. 風險管理（1篇）
4. 操作實務（3篇）
5. CBAS選擇權（4篇）
6. 研究工具（2篇）
7. 實戰案例（6篇）
8. 競拍策略（1篇）

**使用方式：**
- 點擊左側分類列表
- 或使用上方分類下拉選單

### 🏷️ 標籤篩選
- **74個標籤**涵蓋所有主題
- **熱門標籤**：顯示最常用的20個標籤
- **多重篩選**：可同時選擇多個標籤

**標籤範例：**
```
#競拍技巧 #融資警示 #風險控制 #CBAS
#實物履約 #陽明五 #新手入門 #產業分析
```

### 👥 對象篩選
- **新手**：適合剛入門的投資人
- **進階**：適合有一定經驗的投資人
- **專業**：適合深入研究的投資人

### 📊 排序功能
- 日期（新到舊/舊到新）
- 閱讀時間（短到長/長到短）

### 💬 Rex金句庫
- 自動顯示隨機金句
- 每10秒自動更換
- 收錄Rex大叔的經典語錄

### 📖 文章詳細頁
點擊任一文章卡片，會顯示：
- ✓ 完整文章摘要
- ✓ 核心重點整理
- ✓ Rex經典語錄
- ✓ 實用建議
- ✓ 相關標籤
- ✓ 原文連結

## 🐛 問題排解

### 問題1：頁面顯示「載入失敗」
```
原因：找不到 kb_data_complete.json
解決：
1. 確認兩個檔案在同一資料夾
2. 確認檔名正確
3. 重新開啟 HTML 檔案
```

### 問題2：搜尋沒有結果
```
原因：搜尋條件太嚴格
解決：
1. 清除篩選條件
2. 使用更簡短的關鍵字
3. 檢查是否有選擇標籤
```

### 問題3：文章內容顯示不完整
```
原因：彈窗被關閉或未完全載入
解決：
1. 重新點擊文章卡片
2. 等待內容完全載入
3. 點擊「閱讀完整原文」查看方格子文章
```
